# Sales-Performance-Analysis
# 📊 Sales Performance Analysis Dashboard

## Project Overview
This project focuses on analyzing retail sales data to identify trends, performance gaps, and growth opportunities using Power BI.

## 🛠 Tools Used
* **Power BI:** Data Modeling, DAX, and Visualization.
* **SQL:** Initial data cleaning and preprocessing.
* **Excel:** Source data management.

## 📈 Key Insights
* **Top Segment:** The Kids' category was identified as the highest-performing segment.
* **Performance Gap:** Detected a significant sales drop in Q2 (April) due to a lack of promotional discounts.
* **Efficiency:** Automated reporting logic saved approximately 30% of manual processing time.
